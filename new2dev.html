New2Dev Resource Hub

Brief

Problem

Diving into the tech world can initially be overwhleming. Methods to find helpful information on development and design needs to be more organized.

Solution

Design a website that features educational resources for development, design, web hosting, and tech job hunting.

Description

Website featuring resourceful links for developers and digital designers

My Role

Solo Project: UI Design, Development

Tools

WordPress, Photoshop, Sketch

Platform

Desktop, tablet, phone (responsive)

Project Length

~2 months

When

2018 / 2019

Why take on this project?

A couple years ago, I became determined to break into the digital world of coding and UX Design. I have since become a developer and designer. It has been a challenging and rewarding path. There doesn’t appear to be one right or wrong way to establish one’s self in a digital career, but there are certainly methods that are easier than others. Sometimes I’ve learned things the hard (and more time-consuming) way.

It can feel overwhelming in the early stages, trying to figure out what you really need to know. I wanted to provide an informational hub to new developers and designers. Since beginning my own journey, I’ve collected educational resources to help ease into the digital space. I want to share these resources because I wish I’d had them all from the beginning.

This website is for beginner web developers and designers, and anyone else who benefits from having a central, organized hub of helpful web materials.

Challenges

First time with WordPress

The main challenge for this website was using WordPress for the first time. While I usually develop a website from scratch, I wanted to familiarize myself with WordPress as about a third of the internet utilizes this CMS (content management system). I decided to use Elementor, a WordPress Plugin “Page Builder” that helps streamline the website building process. I used the template “Astra” as it fit the layout needs of the site. I chose the cosmic appearance because I felt it represented the element of discovery.

While WordPress is meant to make the website building and monitoring process easier, it does come with a learning curve. There are many dashboard elements that require observation, such as settings, plugins, SEO, Users, Tools, etc. Also, each time a new plugin was utilized meant maintaining all updates for that plugin or the whole site may shut down. Before I became aware of Elementor, even basic construction of my website’s content was unclear to me. Making custom edits with CSS takes some getting used to as it can conflict with existing CSS that comes built into the template.

Resource Organization

Another challenge was providing educational resources in an organized manner that the user would find intuitive and useful. I organized navigation links into different categories such as HTML, CSS, JavaScript, Design, and Job Hunting. I then simply listed all links I had gathered for any one category and placed them on their corresponding pages. This first attempt, while straight-forward and simple, looked very amateur and did a poor job of describing each link. My next iteration (shown below) involves displaying the links as boxes with better descriptions. This way, users can better find what they’re looking for and have a better understanding of the resource before they visit it.

Figure 1: New2Dev Educational Resources Layout

Looking forward, I think there are still possibly better ways to organize and describe the educational resources. Some UX Research featuring interviews and usability tests could greatly improve the user’s ability to find what they need.
